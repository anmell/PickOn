Group Mode

There are [this many] students logged in. How many groups would you like to create?

Instead of a button, the instructor should get to input the number of groups they would like to sort their class into. There should be an algorithm that sorts the names of the students logged in into random groups based on the number of groups chosen by the instructor.

**Click the button to see an example of what would happen if there were 30 students logged in and the instructor chose to sort the class into 6 groups.
